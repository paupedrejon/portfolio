# Skills técnicas priorizadas

Este directorio contiene un catálogo de skills orientadas a proyectos reales de app móvil + backend cloud.

- Archivo principal: `skills-catalog.json`
- Objetivo: tener una base clara y reutilizable para CV, portfolio, roadmap y entrevistas.

## Criterio de selección

Se han incluido y consolidado las áreas más valiosas para un perfil full-stack moderno:

- arquitectura y backend por capas
- móvil con React Native + Expo
- autenticación y seguridad de sesión
- pagos y webhooks
- cloud serverless + almacenamiento
- base de datos y migraciones
- testing + CI/CD + Docker + Git workflows
- calidad de código y configuración por entornos
- consumo de APIs externas y geoespacial
